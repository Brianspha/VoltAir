Detailed Description

A magnetic sensor actor that awards a point when contacted.

Note
Once touched, this actor will change to the player's color and float upwards.

Property Documentation

Body Snowflake::body

Backing physics body for the Actor.

Note
Use of a dynamic property alias is necessitated by C++ property definitions lacking dynamic access to properties in subclasses (e.g. "vertices" in PolygonBody::Body).
